indexOf()를 이용하여 검색하기

HTML 태그 검색하기

  • a : a태그는 다른 페이지 이동을 설정합니다.
  • abbr: abbr태그는 준말이나 머리글자를 나타냅니다.
  • acronym: acronym태그는 줄임말이나 머리글자를 표현하는데 사용합니다.
  • address: address태그는 가까운 HTML요소의 연락처 정보를 나타냅니다.
  • applet: applet태그는 자바 애플릿을 보이게 하는 곳에 쓰입니다.
  • area: area태그는 이미지의 핫스팟 영역을 정의하고 하이퍼링크를 추가할 수 있습니다.
  • article: article태그는 사이트 내에서 독립적으로 구분해 배포하거나 재사용할 수 있는 구획을 나타냅니다.
  • aside: aside태그는 문서의 주요내용, 연관된 부분을 나타냅니다.
  • audio: audio태그는 문서에 소리 콘텐츠를 포함할 때 사용합니다.
  • b: b태그는 독자의 주의를 끌기 위한 용도로 사용합니다.
  • base: base태그는 문서 안의 모든 상대 URL이 사용할 기준 URL을 지정합니다.
  • basefont: basefont태그는 문서의 기본 폰트. 사이즈, 종류를 설정합니다.
  • bdi: bid태그는 텍스트의 출력 방향 영역을 설정합니다.
  • bdo: bdo태그는 텍스트의 방향을 설정합니다.
  • big: big태그는 텍스트의 크기를 설정합니다.
  • blockquote: blockquote태그는 안쪽의 텍스트가 긴 인용문임을 나타냅니다.
