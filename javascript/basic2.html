연산자

01_ 산술 연산자

연산자의 우선순위는 그룹 연산자 '()'를 사용.

++,--연산자는 변수의 앞쪽이나 뒷쪽에 붙여 사용가능

산술 연산자 종류 속성 설명
+ 더하기 연산을 할 때 사용.
문자와 문자, 문자와 변수 등을 연결할 때 사용.
- 빼기 연산을 할 때 사용.
* 곱하기 연산을 할 때 사용.
/ 몫을 구할 때 사용.
++ 1씩 증가시킬 때 사용.
-- 1씩 감소시킬 때 사용.

02_ 대입 연산자

'+='연산자는 숫자가 아닌 문자나 변수 등이 오면 기존 데이터에 새로운 데이터를 연결하여 누적

대입 연산자 종류 설명
= num = 1은 우변의 1값을 좌변의 num 변수에 대합(저장)한다.
+= num += 2는 num = num + 2와 같다.
-= num -= 2는 num = num - 2와 같다.
*= num *= 2는 num = num * 2와 같다.
/= num /= 2는 num = num / 2와 같다.
%= num %= 2는 num = num % 2와 같다.

03_ 비교 연산자

값을 비교하여 true 나 fales로 반환

비교 연산자 종류 설명
> a > b가 참이면 true, 거짓이면 false가 된다.
< a < b가 참이면 true, 거짓이면 false가 된다.
>= a >= b가 참이면 true, 거짓이면 false가 된다.
<= a <= b가 참이면 true, 거짓이면 false가 된다.
== a == b, a와 b가 같으면 true, 같지 않으면 false가 된다.
== 연산자의 경우 데이터 타입은 비교하지 않는다.
ex) 5=='5'는 true
!= a != b, a와 b가 같지 않으면 true, 같으면 false가 된다.
=== a === b, a와 b가 같고 데이터 타입도 같으면 true, 같지 않으면 false가 된다.
ex) 5==='5'는 false
!== a !== b, a와 b가 같지 않거나 데이터 타입이 같지 않지 않으면 true, 같으면 false가 된다.

이상, 초과, 이하, 미만 표현

이상 num >= 10, 10이상을 의미 합니다.
초과 num > 10, 10초과를 의미 합니다.
이하 num >= 10, 10이하를 의미 합니다.
미만 num < 10, 10미만을 의미 합니다.

04_ 논리 연산자

논리 연산자 종류 설명
&& (AND연산자) a && b, a와 b 모두 true일 때 true가 됩니다.
true && true = true
true && false = false
false && true = false
false && false = false
|| (OR연산자) a || b, a와 b 중 하나만 true면 true가 됩니다.
true || true = true
true || false = true
false || true = true
false || false = false
! (NOT연산자) !a, a가 ture면 false, false면 true로 바꾸어 줍니다.
